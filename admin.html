Resumo editorial

O que este painel já entrega

  • Cadastro de avisos para a home com categoria, data e destaque.
  • Gerenciamento de links rápidos da secretaria.
  • Edição do texto principal da home.
  • Rascunho persistido localmente e exportação para publicação futura.

Próximo passo recomendado

Este é um rascunho funcional do painel. O próximo estágio ideal é conectar esse conteúdo a um backend ou fluxo de publicação automatizada para que a secretaria não dependa de exportar arquivos manualmente.

Saída dos dados editoriais

Este painel grava rascunhos localmente no navegador e já alimenta a home neste mesmo ambiente. Para transformar isso em atualização pública permanente no GitHub Pages, o próximo estágio é conectar o painel a um fluxo de publicação.

Enquanto esse backend não existe, você já consegue exportar o conteúdo em JSON. Esse arquivo pode substituir `data/site-content.json` no repositório.

Tudo o que for salvo aqui já pode ser visualizado nesta máquina ao abrir a home do site.

Fluxo manual de publicação

  1. Baixe o JSON gerado neste painel.
  2. Substitua o arquivo `data/site-content.json` no projeto.
  3. Faça commit e envie ao GitHub para atualizar o site público.
