Como publicar conteúdos?

  1. Navegue pelas abas ao lado para editar avisos, alterar links ou imagens.
  2. Tudo que você edita fica salvo automaticamente no seu navegador.
  3. Quando finalizar a revisão do conteúdo, vá na aba Publicação e baixe o arquivo site-content.json finalizado.
  4. Substitua esse arquivo na pasta data/ do projeto no GitHub para a alteração refletir oficialmente para todos.
